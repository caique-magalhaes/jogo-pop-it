# Pop It: Interactive Web Game

An interactive, responsive single-page web game replicating the popular "Pop It" sensory toy. Built using pure vanilla web technologies, the project focuses on lightweight DOM manipulation, state persistence, dynamic event listening, and smooth CSS3.

## 🚀 Live Demo
🎮 **Play the game instantly here:** [caique-magalhaes.github.io/jogo-pop-it/](https://caique-magalhaes.github.io/jogo-pop-it/)

---

## 🛠️ Tech Stack
*   **Core Logic:** JavaScript (ES6+ Vanilla DOM Manipulation)
*   **Structure:** HTML5 Semantic Layouts
*   **Styling & Animation:** CSS3 (Flexbox, and smooth transition properties)
*   **Media Assets:** Integrated sound effects and localized image assets

---

## 💻 Local Setup & Development

To run this game locally on your machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/caique-magalhaes/jogo-pop-it.git
   cd jogo-pop-it
   ```

2. Open the entry point in any browser:
   ```bash
   # On macOS/Linux:
   open index.html
   
   # On Windows:
   start index.html
   ```
